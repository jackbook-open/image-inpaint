Image Inpaint for Windows

Install:
1. Right-click the downloaded zip and choose Extract All.
2. Open the extracted ImageInpaint folder.
3. Double-click Install Image Inpaint.cmd.
4. Open Image Inpaint from the desktop shortcut or Start menu.
5. Click Pre-check before the first processing run.

Portable launch:
You can also double-click ImageInpaint.exe from this folder without installing.

Uninstall:
Open Uninstall Image Inpaint from the Start menu, or double-click
Uninstall Image Inpaint.cmd from this folder.

Responsible use:
Use this app only on documents and images that you own or are explicitly
authorized to modify. Do not use it to remove third-party copyright notices,
attribution, signatures, source marks, or other protected content.

Files:
Original documents and images are not edited in place. The app writes results to
a new output folder with processed images, backups, and a run log.
